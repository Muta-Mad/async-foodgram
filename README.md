# async-foodgram

## Как запустить проект
1. Создайте и активируйте виртуальное окружение.
```
python -m venv venv
source venv/Scripts/activate
```
2. Установите зависимости из файла requirements.txt.
```
pip install -r requirements.txt
```
3. Примените миграции. Находясь в корневом каталоге выполните команду:
```
alembic upgrade heads
```
4. Запустите проект, выполнив команду:
```
python src/main.py
```

## Доступные эндпоинты
Доступные эндпоинты можно посмотреть по адресу:
http://127.0.0.1:8000/redoc

## Запуск скрипта для добавления тегов в БД
Находясь в корневом каталоге выполните команду:
```
python src/management/create_tag
```
