
**[FastCook](http://fastcooke.ru/)** - это современное веб-приложение для поиска, хранения и управления кулинарными рецептами. Проект представляет собой полноценную платформу с системой аутентификации, подписками на авторов, избранным и уникальной функцией "Кулинарная корзина" для суммирования ингредиентов.

## ✨ Ключевые возможности

### 👤 Пользовательская система
- **Регистрация и аутентификация** - создайте свой профиль и управляйте им
- **Система подписок** - следите за понравившимися авторами и их новыми рецептами
- **Персональный профиль** - все ваши рецепты и активность в одном месте

### 📝 Управление рецептами
- **Создание рецептов** - делитесь своими кулинарными шедеврами
- **Редактирование** - вносите изменения в свои рецепты в любой момент
- **Удаление** - управляйте своей коллекцией
- **Просмотр** - удобная карточка рецепта с полным описанием

### ❤️ Взаимодействие
- **Избранное** - сохраняйте понравившиеся рецепты других авторов
- **Лента рецептов** - просматривайте рецепты от авторов, на которых подписаны

### 🛒 Умная корзина ингредиентов
- **Добавление в корзину** - собирайте рецепты для приготовления
- **Автоматический подсчет** - ингредиенты из разных рецептов суммируются автоматически
- **Скачивание списка** - экспорт итогового списка ингредиентов в формате TXT
- **Умное суммирование** - если ингредиент повторяется в разных рецептах, его количество складывается

- ## 🚀 Технологический стек

- **Backend:** Python 3.12+, FastAPI
- **База данных:** PostgreSQL
- **Аутентификация:** fastapi-users
- **Контейнеризация:** Docker, Docker Compose

- ## 🚀 Быстрый старт

### 1. Клонирование репозитория
```bash
git clone git@github.com:Muta-Mad/FastCook.git
cd FastCook
```

### 1. Настройка окружения
```bash
cd backend
```
Создайте файл .env с переменными окружениями по примеру из env.template

### 3. Запуск проекта
```bash
cd ..
docker compose up
```
### 4. Доступ к приложению

🌐 Веб-интерфейс будет доступен по адрессу: http://localhost
